Markdown 语法简介

Markdown 是一种轻量级的标记语言，可以用来快速创建简单的文档。它的语法简单易懂，学习起来也不需要太多的时间。在本文中，我们将介绍 Markdown 的基本语法。

  • 标题

Markdown 中可以使用 1 到 6 个 # 来表示 1 到 6 级标题，例如：

Copy

一级标题

二级标题

三级标题

四级标题

五级标题
六级标题

  • 段落

段落之间需要空一行，可以使用空格或制表符来缩进段落：

这是一个段落。

这是另一个段落。

这是一个缩进的段落。
  • 强调
    使用 * 或 _ 来表示斜体或粗体：

斜体
斜体

粗体
粗体

  • 列表

使用 *、+ 或 - 来表示无序列表，使用数字加 . 来表示有序列表：

Copy

  • 无序列表项 1
  • 无序列表项 2
  • 无序列表项 3
  1. 有序列表项 1
  2. 有序列表项 2
  3. 有序列表项 3

  • 链接

使用 文本 来表示链接：

这是一个 链接。
图片
使用来表示图片：


引用
使用 > 来表示引用：

这是一个引用。

  • 代码

使用 代码 来表示行内代码，使用 代码块 来表示代码块：

这是一段 代码。
